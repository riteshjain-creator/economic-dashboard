# Economic Dashboard

**Purpose:** Daily synthesis of Indian macroeconomic indicators, market movements, and credible news — filtered for signal over noise.

**Philosophy:** Build intelligence over time through feedback loops and historical pattern recognition.

---

## Structure

```
economic-dashboard/
├── index.html              # Public dashboard (GitHub Pages)
├── data/                   # Daily market data snapshots
│   ├── YYYY-MM-DD.json
│   └── history.json        # Aggregated historical data
├── synthesis/              # Daily synthesis notes
│   └── YYYY-MM-DD.md
├── learning/               # System learning files
│   └── priorities.md       # Auto-updated based on Discord feedback
├── sources.md              # Vetted sources list
└── scripts/
    ├── collect-data.js     # Data collection script
    └── generate-dashboard.js # Dashboard generation
```

---

## Updates

**Twice daily:**
- **8:30 AM IST:** Pre-market brief (global overnight, futures, key news)
- **4:30 PM IST:** Post-market synthesis (close, movers, analysis)

**Outputs:**
1. Dashboard updated at GitHub Pages URL
2. Summary posted to Discord #daily-economic-update
3. Synthesis archived in `synthesis/`

---

## Historical Depth

- **1 week:** Day-to-day volatility, news correlation
- **1 month:** Trend analysis, momentum shifts
- **3 months:** Structural changes, policy impacts
- **1 year:** Cyclical context, year-over-year comparisons

---

## Learning System

The system improves over time by:
1. Tracking Discord conversations (what matters to you)
2. Noting which news actually moved markets
3. Refining source credibility based on accuracy
4. Building pattern recognition (correlations, regime changes)
